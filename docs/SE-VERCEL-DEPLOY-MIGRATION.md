## Step 1 тАФ Create the clean copy locally

You have `F:\SE` (existing checkout of `ahmedfawzy8866/SE`). Open **Git Bash** and run:

```bash
cd /f/SE
git pull                                  # get latest, incl. the migration scripts
./scripts/make-clean-copy.sh "/f/SE" "/f/SE Vercel deploy" \
    "https://github.com/sierrablue8866-droid/SE-Vercel-deploy.git"
```

The script copies only tracked source (minus the junk above), writes a clean
`.gitignore`, runs a secret sweep, and makes the first commit on a fresh `main`.
It **stops before pushing** so you can inspect the result.

## Step 2 тАФ Review, then push

```bash
cd "/f/SE Vercel deploy"
git status
git log --oneline           # should be a single clean commit
git ls-files | wc -l        # ~1,142
git push -u origin main
```

> If the GitHub repo was created with a README/license, either start it empty, or
> run `git push -u origin main --force` on this brand-new local history.

## Step 3 тАФ Wire Vercel

1. **Vercel тЖТ Add New тЖТ Project тЖТ Import** `sierrablue8866-droid/SE-Vercel-deploy`.
2. **Root Directory:** leave at repo root (`/`) тАФ it's a monorepo; `vercel.json`
   already targets the right app with
   `pnpm turbo build --filter=sierra-estates-realty...`.
3. **Framework preset:** Next.js (auto-detected). Install command `pnpm install`.
4. **Region:** `fra1` (already set in `vercel.json`).
5. **Environment variables:** add everything from
   `apps/sierra-estates-realty/.env.example` (see checklist below).
6. **Deploy.** The 4 cron jobs in `vercel.json` register automatically.

### Environment variable checklist

Copy the keys from `apps/sierra-estates-realty/.env.example` into Vercel
(Production + Preview). Typical groups:

- **Firebase (client):** `NEXT_PUBLIC_FIREBASE_*`
- **Firebase (admin):** `FIREBASE_SERVICE_ACCOUNT` / `GOOGLE_APPLICATION_CREDENTIALS`
- **Property Finder:** API key + webhook secret
- **WhatsApp / Twilio:** tokens
- **AI:** `GOOGLE_GENERATIVE_AI_API_KEY` (or equivalent)
- **App:** `NEXT_PUBLIC_APP_ENV`, any orchestrate/cron secret

Never paste real secrets into the repo тАФ only into Vercel's env settings.
